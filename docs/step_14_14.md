# Step 14.14: `typeof` / `__typeof__` Operator

## Overview

Add support for `typeof`, `__typeof`, and `__typeof__` type operators. These allow using the type of an expression or another type as a type specifier in declarations.

## Why This Matters

PostgreSQL and GCC headers use `typeof` extensively in macros:

```c
#define Max(a, b) \
    ({ typeof(a) _a = (a); typeof(b) _b = (b); _a > _b ? _a : _b; })
typeof(x) temp = x;
```

## Implementation

### Token

Added `Typeof` variant to `TokenKind`.

### Lexer

Recognizes three spellings:
- `typeof` — GCC extension (and C23 standard)
- `__typeof` — GCC extension
- `__typeof__` — GCC extension (double underscore form)

### Parser

`typeof` is handled in the type-matching section of `parse_type()`:

```rust
TokenKind::Typeof => {
    self.advance();
    self.expect(TokenKind::LParen);
    let ty = if self.is_type_start(&self.current().kind.clone()) {
        // typeof(type) — e.g., typeof(int)
        self.parse_type()
    } else {
        // typeof(expr) — e.g., typeof(x)
        let expr = self.expr();
        self.infer_type(&expr)
    };
    self.expect(TokenKind::RParen);
    ty
}
```

Two forms are supported:
1. **`typeof(type)`** — returns the type directly (e.g., `typeof(int)` → `int`)
2. **`typeof(expr)`** — infers the type of the expression using `infer_type()` (e.g., `typeof(x)` where `x` is `int` → `int`)

Added `Typeof` to `is_type_keyword()` and `stmt()` type-start patterns.

## Type Inference

The `infer_type()` method determines the type of an expression:
- `Num(_)` → `int`
- `FloatLit(_)` → `double`
- `StrLit(_)` → `char *`
- `Var(name)` → lookup in locals/globals
- `Deref(inner)` → base type of pointer
- `Addr(inner)` → pointer to inner's type
- `Cast { ty, .. }` → cast target type
- Other → `int` (fallback)

## Test Cases

```c
int main() { typeof(int) a = 4; return a; }                 // → 4
int main() { int x = 5; typeof(x) y = x; return y; }        // → 5
int main() { return sizeof(typeof(long)); }                  // → 8
int main() { return sizeof(__typeof__(int)); }               // → 4
int main() { int a = 3; __typeof(a) b = a; return b; }      // → 3
```
