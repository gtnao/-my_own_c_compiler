# Step 12.10: Flexible Array Member

## Overview

Support two features related to struct members with array type:

1. **Fixed-size array members**: `struct { int x; int arr[3]; };`
2. **Flexible array member (FAM)**: `struct { int len; int data[]; };`

## Fixed-Size Array Members

Struct members can now be arrays with an explicit size:

```c
struct { int x; int arr[3]; } s;
s.arr[0] = 1; s.arr[1] = 2; s.arr[2] = 3;
return s.arr[1] + s.arr[0];  // => 3
```

The member type is `Array(Int, 3)` with a size of 12 bytes. The struct offset calculation accounts for the array's full size and alignment.

## Flexible Array Member (C99)

A flexible array member has no specified size and must be the last member of a struct:

```c
struct Header {
    int len;
    int data[];   // flexible array member — size 0
};
```

`sizeof(struct Header)` counts only the fixed members (here, 4 bytes for `len`). The flexible array occupies zero bytes in the struct layout but can be accessed via pointer arithmetic when the struct is allocated with extra space:

```c
struct Header *p;
char buf[20];
p = (struct Header *)buf;  // overlay struct on buffer
p->len = 3;
p->data[0] = 42;           // access past the struct's nominal size
```

## Parsing

After parsing the member name, the parser checks for `[`:

```rust
let mem_ty = if self.current().kind == TokenKind::LBracket {
    self.advance();
    if self.current().kind == TokenKind::RBracket {
        // Flexible array: name[]
        self.advance();
        Type::array_of(mem_ty, 0)   // size 0
    } else {
        // Fixed array: name[N]
        let len = /* parse number */;
        self.expect(TokenKind::RBracket);
        Type::array_of(mem_ty, len)
    }
} else {
    mem_ty
};
```

The `Array(base, 0)` type has `size() = base.size() * 0 = 0`, so it doesn't affect the struct's total size. However, the offset is set correctly, so `data[i]` computes `base_addr + offset + i * element_size`, which works for any index when sufficient memory is available.

## Test Cases

```c
// Fixed-size array member
struct { int x; int arr[3]; } s;
s.arr[0]=1; s.arr[1]=2; s.arr[2]=3;
return s.arr[1]+s.arr[0];  // => 3

// Flexible array member via pointer cast
struct { int len; int data[]; } *p;
char buf[20]; p = (void *)buf;
p->len = 3; p->data[0] = 4;
return p->data[0];  // => 4
```
