# ステップ 14.11: `__builtin_*` GCC 組み込み関数

## 概要

よく使われる GCC 組み込み関数のサポートを追加する。これらはパーサー内で特殊ケースとして処理され、コンパイル時に評価されるか、本質的な動作に簡略化される。

## なぜ必要か

PostgreSQL やシステムヘッダでは GCC 組み込み関数が広く使われている:

```c
#define likely(x)   __builtin_expect(!!(x), 1)
#define unlikely(x) __builtin_expect(!!(x), 0)
#define offsetof(type, member) __builtin_offsetof(type, member)
typedef __builtin_va_list va_list;
```

## 実装した組み込み関数

### `__builtin_expect(expr, expected)`

分岐予測ヒント。`expr` をそのまま返す。分岐予測最適化は行わないため、`expected` 引数は無視される。

```c
__builtin_expect(x > 0, 1)  // → (x > 0) として評価される
```

### `__builtin_constant_p(expr)`

`expr` がコンパイル時定数であれば 1 を、そうでなければ 0 を返す。本コンパイラではこの目的で定数式を追跡しないため、常に 0 を返す。

```c
__builtin_constant_p(42)  // → 0
```

### `__builtin_unreachable()`

コード中の到達不能な地点をマークする。ノーオペレーション（0 を返す）としてコンパイルされる。製品レベルのコンパイラではデッドコード除去に利用される。

### `__builtin_offsetof(type, member)`

構造体メンバのバイトオフセットを返す。構造体型のメンバリストを参照してコンパイル時に評価される。

```c
struct S { int a; int b; };
__builtin_offsetof(struct S, b)  // → 4
```

### `__builtin_types_compatible_p(type1, type2)`

2つの型が互換（同じ種類かつ同じ符号性）であれば 1 を、そうでなければ 0 を返す。コンパイル時に評価される。

```c
__builtin_types_compatible_p(int, int)   // → 1
__builtin_types_compatible_p(int, long)  // → 0
```

### `__builtin_va_list`

型名として認識される（`va_list` と等価）。`char*` として扱われる。

### `__builtin_va_start`、`__builtin_va_arg`、`__builtin_va_end`、`__builtin_va_copy`

標準の `va_start`、`va_arg`、`va_end`、`va_copy` マクロのエイリアス。`__va_copy` も処理する。

## 実装

すべての組み込み関数は `primary()` において、識別子が `__builtin_*` の名前に一致した場合に処理される:

- **`__builtin_expect`**: 2つの引数を解析し、最初の引数を返す
- **`__builtin_constant_p`**: 1つの引数を解析し、`Num(0)` を返す
- **`__builtin_unreachable`**: 空の括弧を解析し、`Num(0)` を返す
- **`__builtin_offsetof`**: 型とメンバ名を解析し、構造体メンバからオフセットを参照する
- **`__builtin_types_compatible_p`**: 2つの型を解析し、種類と符号性を比較する

## テストケース

```c
int main() { return __builtin_expect(5, 1); }                              // → 5
int main() { return __builtin_constant_p(42); }                            // → 0
int main() { int x = 42; return __builtin_expect(x, 1); }                  // → 42
struct S { int a; int b; }; int main() { return __builtin_offsetof(struct S, b); } // → 4
struct S { int a; int b; }; int main() { return __builtin_offsetof(struct S, a); } // → 0
int main() { return __builtin_types_compatible_p(int, int); }               // → 1
int main() { return __builtin_types_compatible_p(int, long); }              // → 0
```
