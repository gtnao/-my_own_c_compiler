# Step 14.12: Bitwise Compound Assignment Operators

## Overview

Add support for `&=`, `|=`, `^=`, `<<=`, and `>>=` compound assignment operators. These desugar into `a = a op b` just like the existing arithmetic compound assignments (`+=`, `-=`, `*=`, `/=`, `%=`).

## Why This Matters

PostgreSQL uses bitwise compound assignments extensively for flag manipulation:

```c
flags &= ~MASK;     // clear bits
flags |= FLAG;      // set bits
value <<= shift;    // shift left
hash >>= 1;         // shift right
```

## Implementation

### Token

Added five new token kinds:
- `AmpEq` — `&=`
- `PipeEq` — `|=`
- `CaretEq` — `^=`
- `LShiftEq` — `<<=`
- `RShiftEq` — `>>=`

### Lexer

Added recognition for all five operators:
- `&=` and `|=` are two-character tokens, checked before `&&` and `||`
- `^=` is a two-character token, checked before bare `^`
- `<<=` and `>>=` are three-character tokens, checked before `<<` and `>>`

The ordering is important: `<<=` must be checked before `<<`, and `>>=` before `>>`, otherwise the lexer would consume `<<` and leave `=` as a separate token.

### Parser

Added the five new operators to the compound assignment match in `assign()`:

```rust
let op = match self.current().kind {
    TokenKind::PlusEq => Some(BinOp::Add),
    TokenKind::MinusEq => Some(BinOp::Sub),
    TokenKind::StarEq => Some(BinOp::Mul),
    TokenKind::SlashEq => Some(BinOp::Div),
    TokenKind::PercentEq => Some(BinOp::Mod),
    TokenKind::AmpEq => Some(BinOp::BitAnd),
    TokenKind::PipeEq => Some(BinOp::BitOr),
    TokenKind::CaretEq => Some(BinOp::BitXor),
    TokenKind::LShiftEq => Some(BinOp::Shl),
    TokenKind::RShiftEq => Some(BinOp::Shr),
    _ => None,
};
```

Each desugars `a op= b` into `a = a op b`, reusing the existing binary operator code generation.

## Test Cases

```c
int main() { int a = 3; a &= 1; return a; }           // → 1
int main() { int a = 5; a |= 2; return a; }           // → 7
int main() { int a = 5; a ^= 3; return a; }           // → 6
int main() { int a = 1; a <<= 3; return a; }          // → 8
int main() { int a = 8; a >>= 2; return a; }          // → 2
int main() { int a = 0xFF; a &= 0x03; return a; }     // → 3
```
