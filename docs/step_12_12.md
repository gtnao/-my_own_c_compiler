# ステップ 12.12: _Generic 選択式

## 概要

C11 の `_Generic` キーワードを実装する。これはコンパイル時に制御式の型に基づいて式を選択する:

```c
_Generic(x, int: 1, long: 2, default: 0)
```

`x` が `int` なら結果は `1`、`x` が `long` なら結果は `2`、それ以外は `default` のケースが選択される。

## 構文

```
_Generic(controlling-expression, association-list)
association-list = type-name : assignment-expression
                 | default : assignment-expression
                 | association-list , type-name : assignment-expression
```

## 実装

### パース

`_Generic` は `primary()` でパースされ、パース時に解決される:

1. 制御式をパースする
2. `infer_type()` で型を推論する
3. 型の関連リストを走査する
4. 制御式の型に一致する式、またはマッチしなければ `default` の式を返す

```rust
fn parse_generic(&mut self) -> Expr {
    self.expect(TokenKind::LParen);
    let ctrl_expr = self.assign();
    let ctrl_ty = self.infer_type(&ctrl_expr);
    self.expect(TokenKind::Comma);

    let mut result = None;
    let mut default_expr = None;

    loop {
        if self.current().kind == TokenKind::Default {
            self.advance();
            self.expect(TokenKind::Colon);
            default_expr = Some(self.assign());
        } else {
            let assoc_ty = self.parse_type();
            self.expect(TokenKind::Colon);
            let expr = self.assign();
            if result.is_none() && self.types_match(&ctrl_ty, &assoc_ty) {
                result = Some(expr);
            }
        }
        if self.current().kind == TokenKind::Comma {
            self.advance();
        } else {
            break;
        }
    }
    self.expect(TokenKind::RParen);
    result.or(default_expr).unwrap_or(Expr::Num(0))
}
```

### 型推論（`infer_type`）

式に対するベストエフォートの型推論メソッド:

| 式 | 推論される型 |
|---|---|
| `Num(_)` | `int` |
| `StrLit(_)` | `char *` |
| `Var(name)` | スコープ/グローバルから検索 |
| `Deref(ptr)` | ポインタの基本型 |
| `Addr(expr)` | 式の型へのポインタ |
| `Cast { ty, .. }` | キャストの対象型 |
| その他 | `int`（フォールバック） |

### 型の一致判定（`types_match`）

2つの型が一致するのは、同じ `TypeKind` バリアントと符号属性を持つ場合である。ポインタは基本型が一致すればマッチする。配列とポインタはマッチング目的では互換として扱われる（配列からポインタへの暗黙変換）。

## 使用例

`_Generic` は主に C11 で `<tgmath.h>` のような型ジェネリックマクロを実装するために使用される:

```c
#define abs(x) _Generic((x), \
    int: abs_int, \
    long: abs_long, \
    double: abs_double)(x)
```

## テストケース

```c
int x = 0;  return _Generic(x, int: 1, long: 2, default: 0);   // => 1
long x = 0; return _Generic(x, int: 1, long: 2, default: 0);   // => 2
char x = 0; return _Generic(x, int: 1, long: 2, default: 0);   // => 0 (default)
```
