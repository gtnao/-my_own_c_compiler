# Step 1.6: 比較演算子

## 概要

`==`, `!=`, `<`, `<=`, `>`, `>=` の6つの比較演算子を追加する。パーサーに `equality` と `relational` の2つの優先順位レベルを追加し、コード生成では `cmp` + `setXX` + `movzb` パターンを使う。

## 文法規則の変更

```
expr       = equality
equality   = relational ("==" relational | "!=" relational)*    ← NEW
relational = add ("<" add | "<=" add | ">" add | ">=" add)*     ← NEW
add        = mul ("+" mul | "-" mul)*
mul        = unary ("*" unary | "/" unary | "%" unary)*
unary      = ("+" | "-") unary | primary
primary    = num | "(" expr ")"
```

**優先順位** (低い → 高い): `==`/`!=` < `<`/`<=`/`>`/`>=` < `+`/`-` < `*`/`/`/`%`

## レキサーの変更 — 2文字トークン

`==`, `!=`, `<=`, `>=` は2文字で1つのトークン。`peek_next()` で次の文字を先読みして判定する:

```rust
if ch == '=' && self.peek_next() == Some('=') {
    self.pos += 2;
    tokens.push(Token { kind: TokenKind::EqEq, pos });
    continue;
}
```

`<` と `<=` の区別: `<` の次が `=` なら `Le`、そうでなければ `Lt`。

## コード生成 — cmp + setXX + movzb パターン

比較演算は3命令で実現する:

```asm
  cmp %rdi, %rax     # rax と rdi を比較 (rax - rdi のフラグセット)
  sete %al            # 等しければ al = 1, そうでなければ al = 0
  movzb %al, %rax     # al (8bit) を rax (64bit) にゼロ拡張
```

### 各命令の解説

| 命令 | 動作 |
|------|------|
| `cmp %rdi, %rax` | `%rax - %rdi` を計算しフラグレジスタ (FLAGS) を更新する。結果は破棄 |
| `setXX %al` | FLAGS に基づいて `%al` (raxの下位8ビット) に 0 or 1 をセット |
| `movzb %al, %rax` | `%al` を64ビットにゼロ拡張して `%rax` に格納 |

### setXX バリエーション

| 命令 | 条件 | C演算子 |
|------|------|--------|
| `sete` | ZF=1 (等しい) | `==` |
| `setne` | ZF=0 (等しくない) | `!=` |
| `setl` | SF!=OF (より小さい、符号付き) | `<` |
| `setle` | ZF=1 or SF!=OF | `<=` |
| `setg` | ZF=0 and SF=OF | `>` |
| `setge` | SF=OF | `>=` |

### なぜ movzb が必要か

`setXX` は `%al` (8ビット) にしか書き込まない。`%rax` の上位56ビットには古い値が残っている可能性がある。`movzb` で上位ビットをゼロクリアし、正しい 0 または 1 の値にする。

## テスト

| 入力 | 期待値 | 検証内容 |
|------|--------|---------|
| `1==1` | 1 | 等価 (真) |
| `1==2` | 0 | 等価 (偽) |
| `1!=2` | 1 | 非等価 (真) |
| `1<2` | 1 | 未満 (真) |
| `1<1` | 0 | 未満 (偽: 等しい) |
| `1<=1` | 1 | 以下 (等しい場合も真) |
| `2>1` | 1 | 超過 (真) |
| `2>=2` | 1 | 以上 (等しい場合も真) |
| `5+1==6` | 1 | 算術式と比較の組み合わせ |
| `3*2>=5` | 1 | 乗算と比較の組み合わせ |
