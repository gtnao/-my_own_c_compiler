# ステップ 14.27: 空文（単独のセミコロン）

## 概要

空文（単独の `;`）のサポートを追加する。これにより `for (...);`（空のループ本体）や単独の `;` といったパターンが使用可能になる。

## この実装が重要な理由

PostgreSQLや一般的なCコードでは、空文がいくつかのパターンで使用される:

```c
// Empty for body — all work done in the increment
for (i = 0; i < n; i++);

// Empty while body
while (*p++);

// Standalone semicolons (sometimes from macro expansions)
;
```

## バグの内容

パーサーの `stmt()` 関数には `TokenKind::Semicolon` に対応するケースがなかったため、単独の `;` に遭遇すると式のパースにフォールスルーし、"expected a number, identifier or '('" というエラーが発生していた。

## 修正内容

`stmt()` の先頭に `Semicolon` ケースを追加した:

```rust
TokenKind::Semicolon => {
    self.advance();
    Stmt::Block(vec![])  // empty statement as empty block
}
```

## テストケース

```c
int main() { int i; for (i = 0; i < 10; i++); return i; }    // → 10
int main() { ; return 0; }                                     // → 0
int main() { int i = 0; while (i < 5) { i++; } ; return i; } // → 5
```
